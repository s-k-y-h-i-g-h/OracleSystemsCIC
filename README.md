# Oracle Systems CIC

AI-powered epistemic infrastructure — tools and systems that make information verification, truth-tracking, and evidence assessment accessible to everyone.

## Mission

To build the verification layer the internet is missing: independent, transparent, and accessible to all.

## Products & Projects

### ClaimCheck (in development)
Browser extension + backend that verifies YouTube video titles against multiple independent sources. Uses a tri-source confirmation model — a claim is considered verified when at least three independent sources confirm it.

### OpenClaw Agent Services
Custom AI agent development and deployment for businesses, charities, and public sector organisations. Built on the OpenClaw platform.

## Structure

Oracle Systems CIC is a Community Interest Company. All profits are reinvested in the company's community-purpose activities or transferred to the self++ Foundation (pending charity registration).
